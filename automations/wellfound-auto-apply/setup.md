# Setup Guide (Browser script)

## Prerequisites
- Chrome/Edge browser with console access.
- Values set in the script for applicant details (see env.example).

## Run steps
1. Open Wellfound job listings page.
2. Paste wellfound.js into the console.
3. Set variables per env.example then run.

## Smoke test
- Test on a single listing; ensure application text populates and counters increment.

## Go-live checklist
- Keep run size small to avoid captchas
- Monitor for selector changes after site updates
